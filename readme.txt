Программа может парсить файл с расширением .cs или все файлы .cs в указанной директории без создания PDF файла.

Для проверки видимой проверки корректности программы на консоль выводится полученный элемент и куда он добавился(к какому родителю)

Файлом для запуска является xmldoc2pdf.py

Структура проекта:
1) Классы парсера [Parser], которые выделяют предположительные элементы файла и, после проверки, добавляет их в соответствующий класс-родитель.
2) Классы структуры файла [ProjectStructure], которые проверяют полученные из предыдущего пункта элементы на корректность на столько -- на сколько меня хватило.
3) Утилиты [Tools].
