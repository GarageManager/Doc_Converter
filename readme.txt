Программа может парсить файлы с расширением .cs, .csproj, .sln или всю директорию С# проекта.

Файлом для запуска является xmldoc2pdf.py.

Создается PDF файл документации. За основу оформления взят MSDN.
Используются 2 бесплатных шрифта DejaVuSansCondensed и DejaVuSansCondensed-Bold.

Если в проекте есть синтаксические ошибки, то программа не сможет выдать результирующий файл и выведет сообщение об ошибке на этапе парсинга.

В проекте используется сторонная библиотека FPDF. Используется для формирования шрифта, поддерживающего юникод. Библиотека формирует таблицу глифов используемых символов, таблицу ToUnicode и CIDmap, а так же для вычисления длины строки при использовании какого-либо шрифта
